async-file-dl

A simple asynchronous file downloader based on axios.

Installation

 
$ npm install async-file-dl

Usage

 
const { download } = require('async-file-dl');

download('http://example.com/', '.', 'example.html')
  .then(file => console.log('download successful')
  .catch(console.error);
